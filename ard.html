MIDI Theremin with Arduino

This is my first project in microcontrollers. It's a MIDI instrument that can be played in real time in two ways: as a theremin and with potentiometer knobs. You play with both hands, one controls pitch, the other controls volume. Since it's a MIDI instrument, the notes are quantized and you can choose different modes of playing: major, minor, Lydian, Dorian, Mixolydian, Phrygian. Additional buttons provide extra functionalities, including a recording option.

A challenge i faced was: How do I send MIDI signals from an Arduino Uno board to a DAW? The fastest option I found was through additional software. I used LoopMIDI to create virtual MIDI ports, and Hairless MIDI to send MIDI signals from Arduino to the DAW. For the DAW, I chose Cubase 12 and mapped the controls accordingly.

Components

Arduino Uno board,
2x potentiometers,
LED + resistor,
4 buttons (or more),
2x HC-SR04 ultrasonic sensors,
LED display,
a breadboard.

Code

Input handling

The program reads values from potentiometers and ultrasonic sensors. Using the time-of-flight method, sensor distances are converted into the corresponding note and volume. Buttons are also monitored, with debouncing applied to prevent false triggers and ensure reliable input.

State management

It maintains state variables for the current note, previous note, active scale, and input mode (potentiometer or theremin). Musical scales are stored in a LUT, allowing input values to be quickly mapped to the correct MIDI notes.

MIDI output and feedback

When notes change, the system sends MIDI messages using note-on (0x90) and note-off (0x80) commands. The LED lights up as a visual indicator when recording is active.

Timing and loop structure

The main loop runs continuously, but note and sensor readings are updated at controlled intervals to avoid unnecessary processing and ensure smooth response. Time stamps track the last update for inputs and button presses, which also helps implement debouncing and precise timing for note playback.

Future Improvements / Tips

For future versions of this MIDI theremin, I would using higher-precision sensors. The HC-SR04 sensors used in this project were not very accurate for fine gestures, so better ultrasonic or optical distance sensors could improve responsiveness and playability.
