MATLAB Oscilloscope Simulator


This MATLAB project is a learning tool designed to understand oscilloscope operation and basic signal analysis in both time and frequency domains. It supports two channels and includes features such as MATH operations (addition, subtraction, multiplication, FFT), XY mode (Lissajous patterns), trigger settings (level, slope, source), and dynamic random signal generation. The simulator is ideal for visualizing waveforms and experimenting with oscilloscope functionalities without hardware.

Main Features

Code / Functionality

Signal Generation: Generates random test waveforms dynamically with DC, sine, square, and sawtooth components, each with random frequency, amplitude, and phase.

Trigger Search: Scans the selected channel for points where the waveform crosses the trigger level in the chosen slope direction. If no crossing is found, a random point is chosen to maintain continuous display (mimicking Auto mode).

Math Operations: Real-time CH1 + CH2, CH1 – CH2, CH1 × CH2; FFT for spectral analysis.

Coupling: DC and AC; AC mode removes mean automatically.

Timer & Updates: A MATLAB timer updates every 50 ms. Signals are computed with 1 µs (microsecond) resolution. Maximum visualizable frequency (Nyquist) is calculated as: fNyquist = 1 / (2 × Δt) = 1 / (2 × 1e-6 s) = 500 kHz.

Bonus: Lissajous Patterns
