MATLAB Oscilloscope Simulator

This MATLAB project is a learning tool designed to understand oscilloscope operation and basic signal analysis in both time and frequency domains. It supports two channels and includes features such as MATH operations, XY mode, trigger settings, and dynamic random signal generation.

Code / Functionality

Signal Generation: Generates random test waveforms dynamically with DC, sine, square, and sawtooth components.
Trigger Search: Scans the selected channel for trigger crossings; Auto mode if not found.
Math Operations: CH1 + CH2, CH1 – CH2, CH1 × CH2; FFT for spectral analysis.
Coupling: DC and AC; AC removes mean automatically.
Timer & Updates: Timer updates every 50ms; signals computed with microsecond resolution.
